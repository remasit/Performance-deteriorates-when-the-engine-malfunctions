# Aircraft Performance Prediction (Engine Malfunction)

This project uses Machine Learning to predict aircraft performance when an engine malfunctions.

## Project Files

- `train_model.ipynb` → Model training notebook
- `aircraft_model.pkl` → Saved trained model
- `predict.py` → Script to test the model on new aircraft data
- `requirements.txt` → Required Python libraries

## How to Run

1. Install libraries:
   pip install -r requirements.txt

2. Run prediction:
   python predict.py
